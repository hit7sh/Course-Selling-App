### Course Selling app

Frontend Routes - 
1. /courses - shows all courses
2. /courses/purchased - Shows all purchased courses
3. /courses/:id - Shows a single course, allows user to purchase it
4. /login - login page
5. /signup - signup page
6. / - Landing page
